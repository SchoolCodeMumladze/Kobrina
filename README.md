# Калькулятор с историей запросов
## **Как работать с калькулятором:**
Чтобы работать с калькулятором нужно:

1.Ввести *математическую операцию* 

2.Ввести число(числа),которые вы хотите использовать в действии

3.Получить результат,который автоматическии добавляется в историю запросов

### Для того,чтобы посмотреть историю запросов и закрыть программу,нужно написать команду:**"История"**

## **Какие математические операции может совершать калькулятор:**
-Базовые математические операции(+,-,*,/)

-Нахождение квадратного корня

‼️️*Вызывается командой **"Корень"***

-Возведения в степень

‼️*Вызывается командой **"Степень"***

-Нахождения тригонометрических функций(Синус,Косинус,Тангенс,Котангенс)

-Нахождение факториала числа

‼️*Вызывается командой **"Факториал"***

-Нахождение модуля числа

‼️*Вызывается командой **"Модуль"***

-Округление чисел(Вниз и Вверх)

‼️*Вызывается командой **"Округление"***
